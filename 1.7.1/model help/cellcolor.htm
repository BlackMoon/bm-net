Свойство CellColor

Возвращает или устанавливает цвет сетки.

Синтаксис

object.CellColor
object.CellColor = Color

Параметр Тип Описание
CellColor Long обязательное

Прим.
Используйте макрос [D3DCOLOR_XRGB] для установки цвета.

Пример.

LONG color = app.GetCellColor();
app.SetCellColor(D3DCOLOR_XRGB(255, 0, 0));

Используется объектом Приложение
