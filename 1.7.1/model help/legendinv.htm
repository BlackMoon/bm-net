Свойство LegendInv

Возвращает или устанавливает состояние Инвертировать цвета в цветовой гамме легенды.

Синтаксис

object.LegendInv
object.LegendInv = Inv

Параметр Тип Описание
LegendInv Boolean обязательное

Прим.
При изменении цветовой гаммы требуется создать палитру объекта заново.

Пример.

BOOL inv = app.GetLegendInv();
app.SetLegendInv(FALSE);

Используется объектом Приложение
