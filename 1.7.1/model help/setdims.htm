SetDims

Устанавливает количество отображаемых строк/столбцов/слоев модели.

Синтаксис

object.SetDims(x0, x1, y0, y1, z0, z1)

Параметр Тип Описание
x0 Long обязательное
x1 Long обязательное
y0 Long обязательное
y1 Long обязательное
z0 Long обязательное
z1 Long обязательное

Пример.
Пример показывает с 1 по 20 ряд по X, с 1 по 20 ряд по Y и с 4 по 6 ряд по Z.

graph.SetDims(1L, 20L, 1L, 20L, 4L, 6L);

Используется объектом Граф
