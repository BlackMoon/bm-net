SetLegend

Устанавливает палитру цветов легенды.

Синтаксис

object.SetLegend(Legend)

Параметр Тип Описание
Legend Variant обязательное, массив цветов

Прим.
Используйте макрос [D3DCOLOR_XRGB] для установки цвета.

Пример.

SAFEARRAY* psa = SafeArrayCreateVector(VT_I4, 0, 3);
DWORD* pvdata = (DWORD*)psa->pvData;
pvdata[0] = D3DCOLOR_XRGB(255, 0, 0);
pvdata[1] = D3DCOLOR_XRGB(0, 255, 0);
pvdata[2] = D3DCOLOR_XRGB(0, 0, 255);
COleVariant Legend;
Legend.vt = VT_ARRAY | VT_I4;
Legend.parray = psa;
app.SetLegend(Legend);

Используется объектом Приложение
