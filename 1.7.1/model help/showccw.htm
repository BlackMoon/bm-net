ShowCCW

Устанавливает состояние Показывать сетку/контура/скважины.

Синтаксис

object.ShowCCW(Cell, Cont, Well)

Параметр Тип Описание
Cell Boolean обязательное
Cont Boolean обязательное
Well Boolean обязательное

Пример.
Пример показывает скважины.

graph.ShowCCW(FALSE, FALSE, TRUE);

Используется объектом Граф
