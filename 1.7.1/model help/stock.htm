Подсчет запасов

Позволяет рассчитывать запасы и площадь нефтяной залежи по различным слоям/строкам/столбцам сетки.

Полученный результат может быть сохранен в текстовый файл или передан в другое приложение (напр. MS Excel).

Прим. Для подсчета запасов в файле проекта (см. Подготовка данных) должен присутствовать параметр - Запасы (файл stock.txt)!

Значение параметра запасы в для каждого узла рассчитывается по формуле:
z = m * ntog * s * ro * kper.
z -новый (результирующий) параметр;
m - пористость, ntog - доля коллектора, s - нефтенасыщенность;
ro - плотность нефти, kper - пересчетный коэффициент (если kper > 1, то kper = 1 / kper).
