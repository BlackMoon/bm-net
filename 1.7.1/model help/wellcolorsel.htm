Свойство WellColorSel

Возвращает или устанавливает цвет скважины (выделенный).

Синтаксис

object.WellColorSel
object.WellColorSel = Color

Параметр Тип Описание
WellColorSel Long обязательное

Прим.
Используйте макрос [D3DCOLOR_XRGB] для установки цвета.

Пример.

LONG color = app.GetWetCellColorSel();
app.SetWetCellColorSel(D3DCOLOR_XRGB(128, 128, 128));

Используется объектом Приложение
