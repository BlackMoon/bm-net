Свойство WellFacenameFont

Возвращает или устанавливает основной цвет скважины.

Синтаксис

object.WellFacenameFont
object.WellFacenameFont = Facename

Параметр Тип Описание
WellFacenameFont String обязательное

Пример.

String facename = app.GetWellFacenameFont();
app.SetWellFacenameFont("Tahoma");

Используется объектом Приложение
