Приложение

Объект Приложение вызывает программу Model view.
Управление внутренними объектами Model view осуществляется через объект Приложение.

В VC вызов осуществляется через:

CApplication app;
app.CreateDispatch("Model.application");

Функция CreateDispatch создает новый экземпляр Model view.

Open
Save
Show
SetLegend

Пример.
Запуск приложения и открытие файла модели.

CApplication app;
if (app.m_lpDispatch == 0)
{
app.CreateDispatch("Model.application");
}
app.Show();
app.Open(filename);
app.ReleaseDispatch();
