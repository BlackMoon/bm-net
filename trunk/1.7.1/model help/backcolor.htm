Свойство BackColor

Возвращает или устанавливает цвет фона.

Синтаксис

object.BackColor
object.BackColor = Color

Параметр Тип Описание
BackColor Long обязательное

Прим.
Используйте макрос [RGB] для установки цвета.

Пример.

LONG color = app.GetBackColor();
app.SetBackColor(RGB(255, 0, 0));

Используется объектом Приложение
