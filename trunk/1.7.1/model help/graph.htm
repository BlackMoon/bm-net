Граф

Объект Граф предсталяет 3D визуализацию модели.
Используя методы и свойства объекта Граф возможно изменять отображение текущего состояния модели.

Методы

BackView
BottomView
FrontView
LeftView
ResetFilter
ResetView
RightView
SetDims
ShowCCW
TopView

Пример.
Использование объекта Граф.

CApplication app;
app.CreateDispatch("Model.application");
app.Show();
if (app.Open(filename))
{
LPDISPATH pdisp = app.GetGraph();
if (pdisp)
{
CGraph graph;
graph.AttachDispatch(pdisp);
graph.SetCCW(false, false, true);
}
}
