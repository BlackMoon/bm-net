Свойство LabelColor

Возвращает или устанавливает цвет подписей легенды.

Синтаксис

object.LabelColor
object.LabelColor = Color

Параметр Тип Описание
LabelColor Long обязательное

Прим.
Используйте макрос [RGB] для установки цвета.

Пример.

LONG color = app.GetLabelColor();
app.SetLabelColor(RGB(255, 0, 0));

Используется объектом Приложение
