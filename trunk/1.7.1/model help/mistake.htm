Основные ошибки и способы их устраниения

Модуль ведет запись ошибок и предупреждений, случившихся во время работы в *.log - файл.
Ниже представлен список основных ошибок и способов их устранения.

Код Сообщение Способ устранения

Критические ошибки

Приложению не удалось запуститься, поскольку файл не был найден. Повторная установка приложения может исправить эту проблему. Установите отсутствующий файл
Hardlock module not found. (Error: 7). Установите ключ Hardlock
Can not open Hardlock driver. (Error: 14). Установите драйвер Hardlock
Создание Direct3D завершилось с кодом 0x.. . Отсутствует или выключено ускорение Direct3D. Включите ускорение Direct3D
GetAdapterCount() (нет адаптеров) завершилось с кодом 0x.. . Отсутствует или выключено ускорение Direct3D. Включите ускорение Direct3D
Создание 3D устройства (HAL) завершилось с кодом 0x.. . Отсутствует или выключено ускорение Direct3D. Включите ускорение Direct3D

Ошибки и предупреждения

Не удалось открыть файл. Отсутсвует файл палитры параметра (*.bin). Создайте палитру заново.
Не удалось прочитать значение файл, [секция], значение. Проверьте правильность заполнения значения файл, [секция], значение (*.mvf).
Не удалось прочитать файл, №.. Проверьте правильность заполнения значения файл, № (*.txt).
Не удалось выполнить cbc.exe. Осутствует файл cbc.exe (создание палитры). Повторная установка приложения может исправить эту проблему.
Процесс 0x.. завершился с кодом 0x.. . Установите отсутствующий файл
