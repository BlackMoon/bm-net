Open

Открывает файл проекта (*.mvf), содержащий основные параметры для визуализации.

Синтаксис

object.Open(File, Read)

Параметр Тип Описание
File String обязательное, файл проекта
Read Boolean опция, указывает игнорировать ли ключ bin секции [out] файла проекта

Пример.

app.Open(File, FALSE);

Используется объектом Приложение
