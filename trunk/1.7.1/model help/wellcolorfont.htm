Свойство WellColorFont

Возвращает или устанавливает цвет подписей скважины.

Синтаксис

object.WellColorFont
object.WellColorFont = Color

Параметр Тип Описание
WellColoFontr Long обязательное

Прим.
Используйте макрос [RGB] для установки цвета.

Пример.

LONG color = app.GetWetColorFont();
app.SetWetColorFont(RGB(0, 255, 0));

Используется объектом Приложение
